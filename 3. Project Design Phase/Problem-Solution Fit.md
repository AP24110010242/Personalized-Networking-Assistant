# Problem-Solution Fit

| User Pain Point | App Feature / Solution | Impact |
| --- | --- | --- |
| Conversation anxiety | GPT-2 Icebreaker Generation | Provides immediate conversation starters. |
| Irrelevance to interests | Zero-shot Theme Extraction | Merges event topics with user's specific interests. |
| Factual uncertainty | Wikipedia Fact-checking API | Provides instant, verified text summaries. |
| Forgetting past strategies | History logger (`history.json`) | Allows reviewing successful past conversations. |
| Inflexible suggestions | Feedback logger (`feedback.json`) | Saves rated prompts to refine prompt template. |
