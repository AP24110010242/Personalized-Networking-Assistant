This service module is responsible for the intelligent theme extraction capability of the application. It uses the Hugging Face Transformers pipeline abstraction to load and run the DistilBERT model configured for zero-shot classification. At the module level, when the Python file is first imported, the zero-shot classification pipeline is immediately instantiated. This is an intentional design decision — loading the DistilBERT model into memory once at startup (rather than on every request) means that subsequent requests are processed much faster, since the expensive model-loading step only happens once during the application lifecycle. The extract_event_themes function accepts an event description string and an optional list of candidate labels. If no labels are provided, it defaults to a broad set of professional networking themes such as AI, healthcare, blockchain, education, and sustainability. The DistilBERT zero-shot classifier then scores each candidate label against the input text and returns the top three highest-scoring themes. These themes drive everything that follows — they become the context window for conversation generation.
