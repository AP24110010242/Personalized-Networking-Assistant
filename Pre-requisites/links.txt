Before getting started, ensure you have a basic familiarity with cloud computing and version control (Git) will be helpful. Also, install Visual Studio Code for writing and managing code efficiently.

Python 3.11+: Core programming language used to build the application backend, manage logic, handle APIs, and run AI/NLP processing modules.

https://www.python.org/downloads/

FastAPI Documentation: High-performance Python web framework used to create REST APIs for communication between the frontend, AI services, and verification modules.

https://fastapi.tiangolo.com/

Streamlit Documentation: Interactive frontend framework used to build the web-based user interface for profile input, event descriptions, and displaying generated networking starters.

https://docs.streamlit.io/

Hugging Face Transformers: AI/ML library providing pre-trained NLP models such as DistilBERT and GPT-2 for text classification, theme analysis, and starter generation.

https://huggingface.co/docs/transformers/

Wikipedia API (Python): External verification service used to fetch and validate factual information from Wikipedia sources in real time.

https://pypi.org/project/wikipedia/

Git Version Control: Source code management system used for tracking project changes, collaboration, version history, and deployment workflows.

https://git-scm.com/doc

Virtual Environment (venv): Python environment isolation tool used to manage project dependencies separately and maintain consistent package versions across development setups.

https://docs.python.org/3/library/venv.html
